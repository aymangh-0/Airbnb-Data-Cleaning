# 🧹 Airbnb Data Cleaning & Analysis — Python & Tableau

## Dashboard Preview
![Dashboard](Dashboard_1.png)

## Overview
Data cleaning and analysis project on a large Airbnb NYC dataset containing
102,599 rows and 26 columns, cleaned down to 99,342 rows and 18 columns.

## What Was Done
- Removed irrelevant and duplicate columns
- Removed NaN values
- Renamed and standardized column names
- Standardized text formatting
- Exported clean dataset ready for analysis
- Built an interactive dashboard using Tableau

## Key Insights
- 💰 Average price across all listings is $625
- 📝 Total reviews reached 2,719,206
- 🏠 Total hosts: 99,342
- 🗺️ Manhattan leads in price across all room types
- 🌙 Manhattan has the highest average minimum nights (10 nights)
