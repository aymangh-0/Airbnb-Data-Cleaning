# 🧹 Airbnb Data Cleaning — Python & Pandas

## Overview
Data cleaning project on a large Airbnb NYC dataset containing 102,599 rows
and 26 columns, cleaned down to 99,342 rows and 18 columns.

## What Was Done
- Removed irrelevant and duplicate columns
- Removed NaN values
- Renamed and standardized column names
- Standardized text formatting
- Exported clean dataset ready for analysis

## Files
| File | Description |
|------|-------------|
| `data.ipynb` | Jupyter Notebook with full cleaning steps |
| `clean_data_without_index.csv` | Cleaned dataset |

## Tools Used
- Python — Pandas
- Jupyter Notebook
